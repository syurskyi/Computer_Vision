Guide to Setting Up a Shared Folder on Your Virtual Machine

A shared folder will allow your virtual machine access any folder you specify on your host machine (i.e. Windows or MaOS) within Ubuntu. This will make sharing files between your Host OS and your Virtual Machine much faster and easier.

You will be able to access that folder from Ubuntu's File Manager (see red box below).


Step 1:

Create a folder called “shared-windows10”

Or feel free to use any name you wish.


Step 2:

Click on Shared Folder Settings

This will appear, now Click on the Add New Shared Folder (see circle in red).


Step 3:

Pointing to your shared folder.

When you click the "Add" Button above, this will appear. Now click on the downward point arrow on Folder Path.

This will appear, click on ‘Other…’ A file browser will appear and now just simply select the folder (e.g. My Documents or Downloads or something alled  you wish to share from your host (Windows or MacOS) within your Virtual Machine.


Step 4:

Enable or check the following checkboxes below.


Step 5:

Now go back to Ubuntu and launch a Terminal Window and run the following line:

sudo mount -t vboxsf shared ~/shared-windows10

You will prompted for your machines password (default password I setup was 11223344)

That’s it, you’re done!
