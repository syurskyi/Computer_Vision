To run this Mini Project, either download the  zip file containing the *.ipynb file (in the resources section). It is also included in the full code download, so you may have already downloaded it previously.


If you're not using the Virtual Machine, install the following dependencies:

pip install Pillow
pip install pytesseract

If using Ubuntu, run the following lines in terminal after running those two above:

Remember the VM password is 11223344

sudo apt install tesseract-ocr
sudo apt install libtesseract-dev

If using windows insert this line at the top of your code in the ipython notebook.

pytesseract.pytesseract.tesseract_cmd = r"C:\Program Files (x86)\Tesseract-OCR\tesseract.exe"
